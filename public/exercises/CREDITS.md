# Exercise image credits

The exercise thumbnails in this directory (`<slug>.webp`) are neutral, faceless
line-art figures derived from the **OpenTraining / Everkinetic** exercise image
set:

- Source: https://github.com/chaosbastler/opentraining-exercises
  (originally illustrated by [Everkinetic](https://everkinetic.com/)).
- License: **Creative Commons Attribution-ShareAlike 3.0 (CC-BY-SA 3.0)** —
  https://creativecommons.org/licenses/by-sa/3.0/

**Changes made:** the original PNG line drawings were flattened onto a white
background, letterboxed, resized to ~256px, and re-encoded as WebP via
`scripts/prepare-exercise-images.ts`.

Per CC-BY-SA 3.0, these derived images remain under the same license. Keep this
attribution alongside them if redistributed.
